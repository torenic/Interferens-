Visuell fremstilling

Benyttet beregningsmodell

Diametrisk interferens gjøres om til radial interferens:

δ = Δd / 2

Kontakttrykket finnes fra elastisk utvidelse av boss og elastisk sammentrykking av aksling:

p = δ / { r · [ Cboss + Caksling ] }

Der:

Cboss = (1 − ν²) / E · (b² + r²) / (b² − r²)

Caksling = (1 − ν²) / E · (r² + c²) / (r² − c²)

r = kontakt-radius, b = ytre radius boss, c = indre radius hul aksling. For massiv aksling er c = 0.
